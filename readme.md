# Request to the Yandex Geocode
Пример использование API "Яндекс Геокодер"

Реализованы следующие классы для работы с API:
1) Скрытый класс для хранения API ключей
2) *YandexMaps.py* для структурирования запроса к сервису и сохранения ответа
3) *LocationData.py* для временного хранения данных
4) *DataBaseHandler.py* для работы с SQLite
